Installation

First, add the Chrome Extension from the Google Chrome Store.

And pin it so the icon is visible when you will be navigating.

Configuration with Airtable

Step 1: Create an Airtable Base

To import LinkedIn profiles to Airtable, create an Airtable base with the following columns:

  • Name (Text)
  • LinkedIn Url (Text)
  • Title (Text)
  • Image (Attachment)

Important!

  • Use a Attachment type for the Image field type.
  • Add at least one row of data so the extension will be able to fetch the columns for the matching.

Step 2: Fill out the API Key, Base Id and Table Id

To get your Airtable API key, go to the Help -> API Documentation page. Then, click on the "show API key" checkbox.

Step 3: Map the columns

Once the API Key, the Base Id and the Table Id have been set, you will be able to configure the mapping between the information available for the LinkedIn profiles and your Airtable columns.

Step 4: Go on LinkedIn or refresh the page

If you are already on LinkedIn, refresh the page. If not go to a LinkedIn profile page or a search listing. When the Chrome Extension is well configured, a button will be added to import the profile(s).

Configuration with Google Sheet

Step 1: Sign-in with your Google Account

To import LinkedIn Profiles in a Google Sheet, you will need to authorize the Chrome Extension to edit your Google Spreadsheets.

Step 2: Create a spreadsheet

Create a Google spreadsheet and write following column names on the first row:

  • Name
  • LinkedIn Url
  • Title
  • Image

Step 3: Fill out the spreadsheet id

Then, copy the spreadsheet id visible in the document url with the following pattern: https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit

Step 4: Map the columns

Once the spreadsheet id has been set, you will be able to configure the mapping between the information available for the LinkedIn profiles and your Google spreadsheet columns.

Step 5: Go on LinkedIn or refresh the page

If you are already on LinkedIn, refresh the page. If not go to a LinkedIn profile page or a search listing. When the Chrome Extension is well configured, a button will be added to import the profile(s).
